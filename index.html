Leaflet Bathymetry / Elevation Analysis

This version supports Excel/CSV/TXT upload, UTM to WGS84 coordinate transformation, input point display, IDW-based DTM, initial contours, profile line creation, profile CSV export, hover synchronization between the profile chart and the map, and built-in measurement tools.

1. Upload file

Supported headers include No, X, Y, Z, Z. Elepsoid, Elevation, and Depth. For decimal comma values, .xlsx is the safest format.

2. Input coordinates

If the points appear shifted, the UTM zone is usually the first parameter to check.

3. Point display

4. DTM and contour analysis

Larger grids produce smoother results but are computationally heavier. The contour output here is an initial browser-based result built from a simplified marching-squares approach.

5. Profile and measurement tools

Profile mode lets you draw a line and generate a profile chart. Measurement mode lets you draw a polyline for distance or a polygon for area. Hover on the profile chart to identify the corresponding location on the map.

7. Sample format

For CSV files with decimal commas, use a semicolon separator or save the file as .xlsx.
